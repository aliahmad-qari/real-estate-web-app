# 🏠 PrimeEstate - Modern Real Estate Platform

A professional, full-featured real estate web application built with React and TailwindCSS. Browse premium properties, filter by your preferences, and connect with agents instantly.

## ✨ Features

- 🔍 **Advanced Search & Filters** - Find properties by location, type, price, and purpose
- 🏘️ **Property Listings** - Browse residential, commercial, and luxury properties
- 📱 **Responsive Design** - Seamless experience across all devices
- 🌙 **Dark Mode** - Toggle between light and dark themes
- 💬 **WhatsApp Integration** - Instant contact with property agents
- 📄 **Property Details** - Comprehensive information for each listing
- 🎨 **Modern UI/UX** - Clean, professional design with smooth animations

## 🚀 Tech Stack

- **Frontend:** React 18.2
- **Styling:** TailwindCSS 3.2
- **State Management:** Redux Toolkit
- **Routing:** React Router v6
- **Icons:** React Icons
- **Animations:** Framer Motion

## 📦 Installation

```bash
# Clone the repository
git clone <your-repo-url>

# Navigate to project directory
cd primeestate

# Install dependencies
npm install

# Start development server
npm start
```

The application will open at [http://localhost:3000](http://localhost:3000)

## 🏗️ Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.
